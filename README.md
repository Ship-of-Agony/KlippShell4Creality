# KlippShell 4 Creality

An Android application designed to easily control your **Creality K2 Plus (Klipper)** 3D printer directly from an **Android TV box** (or smartphone/tablet), without needing a PC running constantly.

## Features
* 📱 Optimized for display on Android TV boxes and mobile devices.
* 🔧 Direct access to your printer's Klipper web interface.
* 🖥️ Conveniently manage and monitor your prints right from your TV or couch.

## Installation
To install the app on your Android device:
1. Navigate to the **Releases** section on the right side of this page.
2. Download the latest `.apk` file.
3. Open the file on your Android device to install it (make sure to allow installation from "unknown sources" if prompted).

## Source Code & Development
This project is open-source. Feel free to clone the repository, open it in **Android Studio**, and customize it to your needs.
